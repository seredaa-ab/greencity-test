# GreenCity Tests
## Опис
Цей репозиторій містить тест-кейси та негативні тест-кейси для перевірки сторінки новин GreenCity.
## Посилання на сторінку
https://www.greencity.cx.ua/#/greenCity/events
## Автор
Alina Sereda


## Test Case 1: Перевірка переходу до сторінки новини

**Preconditions:**
Користувач відкрив новину

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1 | Відкрити сторінку | URL | Сторінка відкрилась |
| 2 | Переглянути список новин | - | Відображається список новин |
| 3 | Відкрити новину кнопкою "більше" | - | Відкривається сторінка обраної новини з детальною інформацією: факультет, дата, час, місце, реєстрація, посилання |


## Test Case 2: Перевірка кнопки реєстрації

**Preconditions:**
Користувач реєструється

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1 | Натиснути кнопку "Зареєструватися"  | - | Кнопка на місці, відкрилось вікно реєстрації |
| 2 | Поля вводу наявні | - | Відображаються і вводиться електронна пошта, Ім'я користовача, пароль,  вимоги до пароля під полем пароля, повтор пароля або працює кнопка реєстрації з гугл |
| 3 | Ввести пошту | example@gmail.com | Вводиться, перевіряється чи користувач вже зареєстрован |
| 4 | Ввести пароль | 123456А% | Вводиться, перевіряється на вимоги |
| 5 | Ввести перевірку пароля | 123456А% | Вводиться, значення співрадає з паролем |
