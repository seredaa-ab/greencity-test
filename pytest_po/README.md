# GreenCity UI Automation Tests

## Опис

Автоматизоване тестування UI сторінки подій GreenCity з використанням Selenium, Pytest та Allure.

## Сторінка тестування

https://www.greencity.cx.ua/#/greenCity/events

## Запуск тестів

```bash
pytest
```

## Звіт Allure

```bash
pytest --alluredir=allure-results
allure serve allure-results
```
