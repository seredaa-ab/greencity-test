**Автоматизація тестів для GreenCity
Проект містить автоматизовані тести для перевірки функціоналу сайту GreenCity (https://www.greencity.cx.ua/), включно з подіями, реєстрацією та навігацією через логотип.**
**Посилання:** https://www.greencity.cx.ua/#/greenCity/events
**Інструкція**
1. Клонувати репозиторій:
   git clone <URL вашого репозиторію>
2. Встановити залежності:
   pip install -r requirements.txt
3. Запустити тести:
   python -m unittest discover tests
